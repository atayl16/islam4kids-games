# PWA Icons

This directory contains the Progressive Web App icons for Islam4Kids.

## Required Icons

- `icon-192.png` - 192x192px icon for standard displays
- `icon-512.png` - 512x512px icon for high-res displays and splash screens

## Creating Icons

Icons should:
- Be 192x192px and 512x512px respectively
- Use the Islam4Kids branding colors (green: #2b580c, beige: #f8f4e3)
- Work well as both regular icons and maskable icons
- Be simple and recognizable at small sizes

## Temporary Placeholders

Until custom icons are designed, placeholder icons are used.
Replace these with proper branded icons for production.
